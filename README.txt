📋 Yêu cầu
Trước khi chạy, bạn cần cài đặt các công cụ sau:

Python 3: Đảm bảo bạn đã cài đặt Python 3.6 trở lên.

yt-dlp: Đây là công cụ dòng lệnh cốt lõi. Cài đặt bằng cách chạy lệnh:

pip install yt-dlp

Hoặc tải file thực thi từ trang phát hành chính thức và đặt nó vào cùng thư mục với script hoặc trong PATH hệ thống.

FFmpeg (Rất khuyến khích): Cần thiết để ghép file video và audio, đặc biệt khi tải chất lượng cao. Tải về từ trang chủ FFmpeg và thêm ffmpeg.exe vào PATH hệ thống của bạn.

Cài đặt các thư viện Python cần thiết:

pip install -r requirements.txt
